# Azure AZ-104 Hands-On Project

## Overview
This repository contains hands-on Azure labs based on the AZ-104 curriculum.  
The project demonstrates practical experience in:

- Identity and access management
- Governance and compliance
- Azure resource management
- Networking and connectivity
- Compute and storage
- Monitoring and security fundamentals

All labs were performed using the Azure Portal with documentation and screenshots for verification.

## Why This Project
This project was created to gain real-world Azure administration experience and to demonstrate hands-on skills relevant to:
- Junior Cloud Administrator
- Cloud Support Engineer
- Entry-Level Cloud Security roles

## Tools & Services Used
- Microsoft Azure Portal
- Microsoft Entra ID
- Azure Policy & RBAC
- Virtual Networks & NSGs
- Azure Storage & Virtual Machines
- Azure Monitor & Log Analytics
